## 🚀 Cómo Usar

1. 📝 Ingresa tu mensaje en el campo correspondiente.
2. ⚡ Selecciona "Encriptar" o "Desencriptar".
3. 🎉 ¡Voilà! Tu mensaje ha sido mágicamente cifrado o descifrado.
